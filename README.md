# Astral Gambit Prototype

A small browser-based card game prototype for Session 1 of the development plan. The current build focuses on a clean playable loop: draw cards, spend spark, attack or block, end turns, and resolve a simple win/loss encounter.

## Run locally

```bash
npm install
npm start
```

Open the local Vite URL shown in the terminal.

## Prototype controls

- Click cards in your hand to play them.
- Click **End Turn** to discard your hand and let the enemy act.
- Use **Debug Draw** and **+1 Spark** to quickly test card draw and energy states.
- Click **Restart Encounter** after victory, defeat, or whenever you want a fresh run.
